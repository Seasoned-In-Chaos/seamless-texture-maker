# Unreal Engine 5 Export

SEAMS provides UE5-optimized export with ORM texture packing, DirectX normal format, and automatic material instance generation.

## UE5 Default Settings

| Setting | Value |
|---|---|
| Workflow | Metallic/Roughness |
| Normal Format | DirectX (Y-) |
| Format | PNG |
| Bit Depth | 8-bit |
| Color Management | Renderer Preset |

## UE5-Specific Options

All options default to **enabled**:

| Option | Description |
|---|---|
| ORM Texture Packing | Packs AO (R) + Roughness (G) + Metallic (B) into a single texture |
| Nanite displacement ready | Exports height/displacement with UE5 Nanite compatibility |
| Virtual texture ready | Settings optimized for VT streaming |
| Compression optimized | UE5-compatible compression settings |
| Auto power-of-two validation | Warns if texture dimensions aren't power-of-two |
| Generate UE5 material instance | Creates a text file with material instance parameters |
| sRGB auto assignment | Correctly flags sRGB vs linear textures |

## ORM Packing

When ORM packing is enabled, three maps are combined into a single RGB texture:

| Channel | Map | Source |
|---|---|---|
| **R** | AO | Ambient Occlusion map |
| **G** | Roughness | Roughness map |
| **B** | Metallic | Metallic map |

This reduces texture memory from 3 separate textures to 1, which is the standard UE5 practice.

## File Naming

UE5 uses the `T_` prefix convention:

```
T_MaterialName_D.png      — Diffuse/BaseColor
T_MaterialName_N.png      — Normal Map
T_MaterialName_ORM.png   — Packed ORM
T_MaterialName_Height.png — Height/Displacement
```

## Material Instance

When "Generate UE5 material instance" is enabled, a text file is created:

```
MaterialName_UE5_material_instance.txt
```

<Tip>
For best results in UE5, import the ORM texture with sRGB disabled and compression set to "UserInterface2D (RGBA)". Set the BaseColor texture to sRGB enabled.
</Tip>
