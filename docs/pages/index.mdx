## Why SEAMS?

Traditional seamless texture tools either destroy edge details with Gaussian blur or require manual Photoshop work. SEAMS solves this with:

- **Distance-based gradient blending** — Smooth falloff at seams without losing detail
- **Splat synthesis** — Texture bombing for organic, non-repetitive patterns
- **Chunked processing** — Automatic tiling for images over 2048px with corner artifact elimination
- **Live preview** — 3-stage progressive preview at 12.5%, 50%, and 100% resolution
- **Result caching** — LRU cache avoids redundant recomputation across parameter changes
- **Auto-update** — SHA256-verified updates downloaded and applied on restart

## System Requirements

| Requirement | Minimum | Recommended |
|---|---|---|
| **OS** | Windows 10 (64-bit) | Windows 11 |
| **Python** | 3.12 | 3.12 |
| **RAM** | 8 GB | 16 GB+ |
| **GPU** | Any (CPU fallback) | NVIDIA with CUDA |
| **OpenGL** | 2.1 | 4.5+ |
| **Disk** | 500 MB | 1 GB+ |

<Note>
SEAMS runs on CPU only — no GPU is required. CUDA acceleration is optional and provides 3–5x speedup for heavy operations.
</Note>
