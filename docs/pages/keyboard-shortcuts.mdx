# Keyboard Shortcuts

## File Operations

| Shortcut | Action |
|---|---|
| `Ctrl+O` | Open Texture |
| `Ctrl+S` | Save Current Texture |
| `Ctrl+Shift+S` | Save As... |
| `Ctrl+E` | Export Selected Map |
| `Ctrl+Shift+E` | Export Renderer Pipeline |
| `Alt+F4` | Exit |

## Navigation

| Shortcut | Action |
|---|---|
| `1` | Switch to Delight mode |
| `2` | Switch to Seamless mode |
| `3` | Switch to Material Lab mode |

## Viewport

| Shortcut | Action |
|---|---|
| `Ctrl+0` | Reset Viewport / Fit to View |
| `Escape` | Exit Fullscreen (3D viewport) |
| Scroll wheel | Zoom in/out |
| Left-click drag | Pan (2D) / Orbit (3D) |
| Right-click drag | Pan (3D) |
| Double-click (3D) | Reset camera |

## Edit

| Shortcut | Action |
|---|---|
| `Ctrl+Z` | Undo (up to 20 steps) |
| `Ctrl+Y` | Redo (up to 20 steps) |

## Help

| Shortcut | Action |
|---|---|
| `F1` | Show Keyboard Shortcuts dialog |

## Drag & Drop

| Action | Behavior |
|---|---|
| Drop image file on 2D viewport | Opens the image |
| Double-click empty viewport | Opens file dialog |
